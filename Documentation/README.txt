pour le generateur de labyrinthe :
metre une image en parametre Map du Generator, un modele est deja present dans le dossier maps
pour qu'une image puisse etre utilisée pour creer un labyrinthe, il faut que ses parametres dans unity soient configurés ainsi:
Filter Mode : Point (no filter)
Compression : None
dans Advanced :
Non Power of 2 : None
Read/Write Enabled : oui
